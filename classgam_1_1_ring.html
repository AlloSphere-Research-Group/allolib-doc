Detailed Description

template<class T, class A = gam::Allocator<T>>
class gam::Ring< T, A >

Ring buffer.

This is a circular buffer that permits sequential writing and reading of elements in an array up to some number of past elements.

Template Parameters
Tarray element type
Amemory allocator
