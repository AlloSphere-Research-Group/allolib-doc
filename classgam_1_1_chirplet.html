Detailed Description

template<class Tv = double, class Td = DomainObserver>
class gam::Chirplet< Tv, Td >

Swept sinusoid with Gaussian envelope.

This generates a sinusoid with a linear frequency sweep and Gaussian envelope. Only two complex multiplies are required per sample. Single-precision should be used only for very short envelopes due to accumulation error.
