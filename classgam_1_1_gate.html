Detailed Description

template<class T = real, class Td = DomainObserver>
class gam::Gate< T, Td >

Binary gate controlled by threshold comparison.

The gate returns 1 if the input magnitude is above a specified threshold, otherwise it returns 0. A closing delay can also be specified to determine the window of time the input must be below the threshold before the gate closes. This is equivalent to what is known in electronics as a comparator with hysteresis.
