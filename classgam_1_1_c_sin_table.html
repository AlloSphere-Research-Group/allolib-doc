Detailed Description

template<unsigned B = 10, unsigned D = 2, class TComplex = Complex<double>>
class gam::CSinTable< B, D, TComplex >

Complex sinusoid lookup table.

This retrieves a complex sinusoid e^it given a phase t. D tables each with a size of 2^B are used to compute the value. D-1 tables are used for recursive interpolation. This makes the lookup equivalent to a non-interpolating lookup in a table of size (2^B)^D.

Template Parameters
Blog2 size of each table (size == 2^B)
Dthe number of tables
