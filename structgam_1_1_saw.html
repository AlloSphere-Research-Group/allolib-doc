Detailed Description

template<class Tv = gam::real, class Td = DomainObserver>
struct gam::Saw< Tv, Td >

Band-limited saw wave.

This produces a Fourier representation of a saw wave where the number of harmonics is adjusted automatically to prevent aliasing. Due to numerical issues, this generator should not be used for producing very low frequency modulation signals. For that purpose, it is better to use the LFO class.
