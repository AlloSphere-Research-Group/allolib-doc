Detailed Description

template<class Tv = gam::real>
class gam::ZeroCrossRate< Tv >

Computes the zero-crossing rate of an input signal.

The zero-crossing rate (ZCR) is a measure proportional to how many times a signal crosses zero over a given number of samples. It can be used to distinguish between noise (high ZCR) and tones (low ZCR).
