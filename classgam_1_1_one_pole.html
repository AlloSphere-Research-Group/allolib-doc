Detailed Description

template<class Tv = gam::real, class Tp = gam::real, class Td = DomainObserver>
class gam::OnePole< Tv, Tp, Td >

One-pole filter.

This filter uses a single pole at either DC or Nyquist to create a low-pass or high-pass response, respectively.
