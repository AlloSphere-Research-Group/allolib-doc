bool gam::SoundFile::openWrite ( )

Open sound file for writing.

Before calling this method it is necessary to set the number of channels and frame rate. If no EncodingType is set, then PCM_16 is used. The Format of the sound file is derived from the extension.

Returns
true on success and false otherwise.
