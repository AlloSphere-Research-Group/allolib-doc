Detailed Description

template<class T, class A = gam::Allocator<T>>
class gam::DoubleRing< T, A >

Double buffered ring-buffer.

This is a two-part buffer consisting of a ring buffer for writing and a standard (absolute indexed) array for reading.

Template Parameters

Amemory allocator
