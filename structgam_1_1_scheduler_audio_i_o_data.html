void gam::SchedulerAudioIOData::mapAudioIOData ( TAudioIOData &  externalIO)

Map external to internal audio I/O data.

Use this method to map an external data structure holding audio I/O data (buffers, frame rate, block size, number of channels, etc.). This method exists primarily to make the Scheduler easy to use in conjunction with gam::AudioIOData.
