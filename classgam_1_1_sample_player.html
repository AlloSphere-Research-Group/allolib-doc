Detailed Description

template<class T = float, template< class > class Si = ipl::Trunc, class Sp = phsInc::OneShot>
class gam::SamplePlayer< T, Si, Sp >

Sample buffer player.

This streams a sequence of frames from a n-channel buffer according to a specified playback rate. Minimum and maximum endpoint frames are supported for playing back a subinterval of the buffer. The number of frames in the sample should not exceed 2^32. This equates to 27 hours at 44.1 kHz.

Template Parameters
TValue (sample) type
SiInterpolation strategy
SpPhase increment strategy
