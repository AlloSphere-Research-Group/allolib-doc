virtual void gam::DFT::inverse ( float *  dst = 0)
virtual

Performs inverse transform on internal spectrum.

The resynthesized samples are copied into 'dst'. The destination array must have room for at least sizeDFT() number of elements. If 'dst' equals 0, then the resynthesized samples are not copied, but instead held in the internal inverse transform buffer.

Reimplemented in gam::STFT.

Set number of real-valued auxiliary buffers.

Each buffer allocated will maintain a size equal to the number of bins. Memory is guaranteed to be contiguous. This means if you need a complex-valued buffer, you can simply allocate two auxiliary buffers and use the address of the first one as the complex-valued buffer.
